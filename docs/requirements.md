# Requirements

## Goal

Побудувати персональну систему аналізу читання: не просто список книг, а модель читацького смаку.

## MVP

- Зберігати книги, авторів, серії, жанри.
- Зберігати статус читання: to-read, currently-reading, paused, read, dnf.
- Зберігати оцінку, сторінки, дату початку й завершення.
- Зберігати обговорення книги.
- Зберігати прогноз, наскільки книга сподобається.
- Рахувати базову статистику.
- Тримати Reader DNA і гіпотези.
